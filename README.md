## 🏗️ Landing Page - Construction

## 📌 Sobre o Projeto

Esta landing page foi desenvolvida com foco no setor da construção civil, com o objetivo de apresentar serviços, projetos e captar clientes de forma estratégica.

O layout foi planejado para transmitir credibilidade, profissionalismo e confiança, guiando o usuário de maneira intuitiva até a ação principal (CTA), como solicitação de orçamento ou contato direto.

## 🎯 Objetivo

A página tem como principal objetivo converter visitantes em potenciais clientes, utilizando uma estrutura clara, direta e otimizada para tomada de decisão.

Landing pages são criadas justamente com esse foco: direcionar o usuário para uma única ação, reduzindo distrações e aumentando as chances de conversão .

## 🧩 Estrutura da Página

A landing page segue uma estrutura moderna e eficiente:

🏠 Header – Identidade visual e navegação simples
🏗️ Seção Principal (Hero) – Destaque para serviços e proposta de valor
📄 Sobre / Serviços – Apresentação clara dos diferenciais
🖼️ Projetos – Demonstração visual de trabalhos realizados
⭐ Diferenciais – Pontos fortes da empresa
📞 Contato / CTA – Chamada para ação estratégica

Cada elemento foi pensado para guiar o usuário até a conversão, seguindo boas práticas de design e experiência do usuário .

## 🎨 Design e Experiência

O design utiliza elementos visuais fortes, cores sólidas e tipografia marcante, transmitindo a identidade do setor da construção.

Além disso, a página é:

📱 Responsiva – Adaptada para todos os dispositivos
⚡ Rápida e objetiva – Foco na experiência do usuário
🎯 Direcionada – Sem distrações desnecessárias

## 🚀 Tecnologias Utilizadas

HTML5 🟧
CSS3 🟦
JavaScript 🟨

## 💡 Considerações

Este projeto demonstra a importância de uma landing page bem estruturada para empresas da construção civil, sendo uma ferramenta essencial para geração de leads e aumento de resultados no ambiente digital .
